THE PROBLEM

투자의 단계마다
필요한 판단은 서로 달랐다.

종목을 찾을 때는 새로운 촉매가 필요하고, 매수 전에는 가격과 근거가 필요하며, 보유 후에는 대응 기준이 필요합니다. 시장이 움직인 뒤에는 그 원인을 이해해야 다음 판단이 나아집니다. 기존 서비스는 이 과정을 서로 다른 화면과 정보원으로 분리했습니다.

비밀노트는 투자 전 과정을 하나의 학습 가능한 의사결정 흐름으로 연결했습니다.

USER JOURNEY

탐색에서 대응과
시장 학습까지

  1. 01 · DISCOVER신규 종목을 탐색한다

    다가오는 이벤트와 실시간 트렌드에서 영향 업종과 종목 후보를 찾습니다.

  2. 02 · DECIDE매수 타이밍을 판단한다

    종목 상세의 요약·전략·근거를 따라 진입 조건과 판단 근거를 검토합니다.

  3. 03 · MANAGE보유종목에 대응한다

    관심종목을 보유종목으로 보고 장 상황에 맞는 모니터링 순서와 대응 기준을 확인합니다.

  4. 04 · LEARN시장 원리를 학습한다

    금리·달러·채권·원자재·위험자산이 국내 업종과 종목에 전달되는 경로를 이해합니다.

트렌드 분석 · 신규 종목 탐색

01 / DISCOVER

Trend Discovery

신규 종목 탐색

USP다가오는 이벤트를 업종과 종목의 새로운 탐색 기회로 변환합니다.
WHY
등락률 순위나 익숙한 종목만 보면 이미 움직인 가격을 뒤따르기 쉽고, 앞으로 시장을 움직일 촉매를 놓치게 됩니다.
DATA USED
경제 이벤트 캘린더, 원유·환율·금리 축, 실시간 시장 뉴스, 영향 변수·업종, 대표 종목과 이벤트 중요도를 사용합니다.
USER JOURNEY
주간 주요 이벤트를 확인하고 영향 흐름을 펼쳐 수혜·부담 업종과 종목 후보를 찾은 뒤 종목 상세로 이동합니다.
EXPERIENCE
발표 시각, 중요도, 예상 영향, 확인 지표와 인과 흐름을 카드와 영향 그래프로 연결해 보여줍니다.
EXPECTED IMPACT
인기 종목 중심의 탐색 편향을 줄이고, 이벤트가 가격에 반영되기 전 구조적으로 후보를 발견할 수 있습니다.
  • 이벤트 축 분류
  • 업종·종목 영향 매칭
  • 수혜·부담 시나리오

종목 상세 · AI 요약

02 / DECIDE · SUMMARY

Signal Summary

매수 판단 요약

USP현재 흐름과 7개 데이터 축을 함께 보여줘 매수 검토의 출발점을 만듭니다.
WHY
당일 급등만 보면 추격 매수하기 쉽고, 중기 약세나 수급 매도를 따로 보면 기회를 놓칠 수 있어 신호의 충돌을 함께 봐야 합니다.
DATA USED
KIS 실시간 시세, 1·3개월 가격 이력, 거래대금, 외국인·기관 수급, 밸류·실적, 뉴스 감성을 정규화합니다.
USER JOURNEY
후보 종목을 열어 현재가와 추세 지속력을 확인하고, 긍정·부정 신호와 분석 데이터 완성도를 비교합니다.
EXPERIENCE
현재가를 최상단에 두고 긍정은 빨강, 부정은 파랑으로 구분하며 종합 판단과 핵심 신호를 한 화면에 요약합니다.
EXPECTED IMPACT
한 가지 지표에 대한 과신과 무리한 추격 매수를 줄이고, 다음 전략 검토가 필요한 종목을 빠르게 구분할 수 있습니다.
  • 7개 데이터 축 정규화
  • 장중 맥락 반영
  • 신호 충돌 설명

종목 상세 · AI 전략

03 / DECIDE · STRATEGY

Entry Strategy

매수 타이밍 전략

USP종목 판단을 관찰 가격대와 진입 조건으로 변환해 매수 타이밍을 구체화합니다.
WHY
좋은 종목이라는 판단과 지금 사도 되는 가격이라는 판단은 다르므로, 가격 조건이 없으면 매수 순간에 다시 감정적으로 결정하게 됩니다.
DATA USED
현재가·전일 종가, 변동성, 단기·중기 모멘텀, 거래대금 변화, 가격 분포와 분석 데이터 완성도를 사용합니다.
USER JOURNEY
신규 매수 보류·관찰 여부를 먼저 확인하고 관찰 가격대, 돌파 조건과 무효화 가격을 순서대로 검토합니다.
EXPERIENCE
‘오늘의 판단 → 관찰 가격대 → 전환 조건 → 축소 기준’ 구조와 가격 기준선으로 긴 분석보다 실행 조건을 먼저 보여줍니다.
EXPECTED IMPACT
충동적인 추격 매수를 줄이고, 진입 전에 기대 수익과 손실 기준을 함께 세우는 습관을 만들 수 있습니다.
  • 행동 우선 정보 구조
  • 동적 가격 시나리오
  • 근거 점진 공개

종목 상세 · 근거 검증

04 / DECIDE · EVIDENCE

Evidence Review

판단 근거 검증

USP매수 판단에서 최신 리포트와 원문 출처까지 한 흐름으로 추적합니다.
WHY
매수 전에는 AI의 결론보다 근거의 최신성과 출처가 더 중요하므로, 판단이 어떤 데이터에서 나왔는지 추적 가능해야 합니다.
DATA USED
증권사 리포트 목록·상세·PDF, 목표주가, 투자의견, 발행일, DART 공시와 뉴스 원문 링크를 종목코드에 연결합니다.
USER JOURNEY
요약과 전략을 검토한 뒤 근거 탭에서 최신 리서치 지표를 확인하고 필요한 문서는 원문까지 열어 직접 검증합니다.
EXPERIENCE
최신 리포트 수, 목표가와 의견을 요약하고 발행사·날짜·제목·원문 링크를 스캔 가능한 목록으로 제공합니다.
EXPECTED IMPACT
매수 판단의 설명 가능성과 신뢰를 높이고, 오래되었거나 출처가 불명확한 정보가 결정에 섞이는 위험을 줄일 수 있습니다.
  • 5분 증분 수집
  • 일일 과거자료 보강
  • 종목·리포트 자동 연결

관심종목 · 보유종목 대응 브리핑

05 / MANAGE

Holdings Response

보유종목 대응

USP관심종목을 보유종목으로 보고 장 상황에 맞는 대응 우선순위를 생성합니다.
WHY
매수 이후에는 새로운 종목 추천보다 내 종목에 영향을 주는 사건과 손실·기회 요인을 빠르게 확인하는 것이 더 중요합니다.
DATA USED
관심종목 구성, 실시간 시세, 외국인·기관 수급, 종목 뉴스, 주요 이벤트, 미국 지수·섹터 흐름과 종목별 민감도를 사용합니다.
USER JOURNEY
국내 장전·장중·장마감과 미국장 상태에 따라 AI 시황 리포트를 확인하고 모니터링 우선 종목부터 대응합니다.
EXPERIENCE
핵심 시황 한 줄, ‘지금 확인’, 위험 신호, 이벤트·뉴스 요약과 보유종목 우선순위를 짧은 리포트로 제공합니다.
EXPECTED IMPACT
보유종목과 무관한 시황 탐색을 줄이고, 중요한 변화가 생긴 종목부터 일관된 기준으로 대응할 수 있습니다.
  • 장 상태 인식
  • 종목 영향도 매칭
  • 모니터링 우선순위

시장 영향도 분석 · 금융 지식 학습

06 / LEARN

Market Learning

시장 영향 학습

USP외부 변수가 국내 업종과 종목으로 전달되는 인과관계를 시각적으로 설명합니다.
WHY
시장 뉴스의 결과만 외우면 다음 사건에 적용하기 어렵기 때문에, 금리·환율·원자재 변화가 주가에 전달되는 원리를 이해해야 합니다.
DATA USED
미국 국채금리·실질금리, 원/달러·달러지수, 장단기 금리차, VIX, WTI·구리, 나스닥과 위험자산 지표를 사용합니다.
USER JOURNEY
다섯 외부 요인의 방향과 비중을 보고 인과 흐름, 영향 업종, 대표 종목과 공식 지표를 차례로 확인합니다.
EXPERIENCE
요인 맵, ‘외부 변수 → 국내증시 → 영향 종목’ 흐름과 학습 포인트를 결합하고 데이터 연결 상태도 투명하게 표시합니다.
EXPECTED IMPACT
시장 변화를 독립적으로 해석하는 금융 지식을 쌓고, 새로운 뉴스에도 같은 인과 구조를 적용할 수 있습니다.
  • 5요인 영향도
  • 인과관계 시각화
  • 공식 지표 추적
