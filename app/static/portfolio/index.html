THE PROBLEM

정보는 많지만,
지금 해야 할 행동은 보이지 않았다.

개인 투자자는 시세 앱, 뉴스, 공시, 증권사 PDF와 미국 시장을 따로 확인합니다. 정보는 계속 늘어나지만 내 종목에 어떤 영향을 주는지, 지금 무엇을 모니터링해야 하는지는 스스로 다시 해석해야 합니다.

비밀노트는 이 해석 비용을 줄이는 데서 출발했습니다.

USER JOURNEY

시장 변화에서
검증 가능한 행동까지

  1. 01 · DISCOVER변화를 감지한다

    오늘 시장에서 달라진 신호와 주요 이벤트를 찾습니다.

  2. 02 · UNDERSTAND내 종목에 연결한다

    시장 변화가 관심종목에 미치는 방향과 강도를 해석합니다.

  3. 03 · ACT대응 기준을 세운다

    관찰 가격과 대응 순서를 확인해 감정적 결정을 줄입니다.

  4. 04 · VERIFY근거를 검증한다

    리포트, 공시와 뉴스 원문으로 판단의 최신성을 확인합니다.

관심종목 · AI 시황 리포트

01 / CONTEXT

Market Intelligence

AI 시황 브리핑

USP시장 이벤트를 개인 관심종목의 모니터링 우선순위로 변환합니다.
WHY
거시 지표와 해외 증시 뉴스가 내 종목에 실제로 중요한지 판단하려면 여러 서비스를 오가며 다시 해석해야 했습니다.
DATA USED
경제 이벤트 캘린더, 시장 뉴스와 감성, 미국 지수·섹터 흐름, 외국인·기관 수급, 사용자 관심종목을 함께 사용합니다.
USER JOURNEY
접속 즉시 오늘의 핵심 변수와 영향받을 관심종목을 확인하고, 우선순위가 높은 종목부터 점검합니다.
EXPERIENCE
한 줄 시황, ‘지금 확인’ 요약, 위험 신호 배지와 모니터링 순위로 복잡한 시장 정보를 압축해 보여줍니다.
EXPECTED IMPACT
시황 탐색 시간을 줄이고, 내 종목과 관련성이 낮은 이슈에 과도하게 반응하는 행동을 줄일 수 있습니다.
  • 이벤트·뉴스 의미 분류
  • 종목 영향도 매칭
  • 모니터링 우선순위 생성

종목 상세 · AI 요약

02 / SYNTHESIS

Stock Diagnosis

멀티팩터 종목 진단

USP서로 충돌하는 7개 데이터 축을 하나의 판단 언어로 설명합니다.
WHY
당일 급등, 중기 약세, 수급 매도처럼 신호가 충돌할 때 단일 확률만 제시하면 현재 흐름과 판단 사이에 괴리가 생깁니다.
DATA USED
KIS 실시간 시세, 1·3개월 가격 이력, 거래대금, 외국인·기관 수급, 밸류·실적, 뉴스 감성을 정규화합니다.
USER JOURNEY
종목을 검색한 뒤 현재 흐름을 먼저 보고, 상승을 지지하는 신호와 경고하는 신호를 비교해 판단의 맥락을 이해합니다.
EXPERIENCE
현재가를 최상단에 두고 긍정은 빨강, 부정은 파랑으로 구분하며 핵심 신호와 분석 데이터 완성도를 함께 표시합니다.
EXPECTED IMPACT
한 가지 지표에 대한 과신과 추격 매수를 줄이고, 데이터가 부족한 판단은 보수적으로 받아들일 수 있습니다.
  • 7개 데이터 축 정규화
  • 장중 맥락 반영
  • 신호 충돌 설명

종목 상세 · AI 전략

03 / ACTION

Action Strategy

조건 기반 대응 전략

USP분석 결과를 관찰 가능한 가격 조건과 대응 순서로 번역합니다.
WHY
좋은 분석도 다음 행동이 명확하지 않으면 읽어야 할 정보만 늘어나고 실제 투자 순간에는 다시 감정적으로 결정하게 됩니다.
DATA USED
현재가·전일 종가, 변동성, 단기·중기 모멘텀, 거래대금 변화, 가격 분포와 분석 데이터 완성도를 사용합니다.
USER JOURNEY
오늘의 대응을 먼저 확인하고 관찰·전환·축소·이익 관리 가격을 순서대로 본 뒤 필요할 때만 상세 설명을 펼칩니다.
EXPERIENCE
‘오늘의 대응 → 핵심 가격 → 대응 순서’ 구조와 점진적 정보 공개로 긴 분석보다 실행 기준을 먼저 보여줍니다.
EXPECTED IMPACT
충동적인 추격 매수와 기준 없는 손절을 줄이고, 동일한 조건에는 일관된 방식으로 대응하도록 돕습니다.
  • 행동 우선 정보 구조
  • 동적 가격 시나리오
  • 근거 점진 공개

종목 상세 · 근거 검증

04 / EVIDENCE

Evidence Layer

출처 기반 근거 검증

USPAI 판단에서 최신 리포트와 원문 출처까지 한 흐름으로 추적합니다.
WHY
투자 판단에서 AI의 문장만으로는 최신성과 신뢰도를 확인하기 어렵기 때문에 결론이 어떤 근거에서 나왔는지 추적 가능해야 합니다.
DATA USED
증권사 리포트 목록·상세·PDF, 목표주가, 투자의견, 발행일, DART 공시와 뉴스 원문 링크를 종목코드에 연결합니다.
USER JOURNEY
AI 요약에서 근거 탭으로 이동해 최신 리서치 지표를 확인하고, 필요한 문서는 원문까지 열어 직접 검증합니다.
EXPERIENCE
최신 리포트 수, 목표가와 의견을 요약하고 발행사·날짜·제목·원문 링크를 스캔 가능한 목록으로 제공합니다.
EXPECTED IMPACT
설명 가능성과 신뢰를 높이고, 오래되었거나 출처가 불명확한 정보가 판단에 섞이는 위험을 줄일 수 있습니다.
  • 5분 증분 수집
  • 일일 과거자료 보강
  • 종목·리포트 자동 연결
