문제 발견

AI 콘텐츠는 많았지만,
투자 판단의 여정은 끊겨 있었다.

실시간 이슈에서 종목을 발견하고, 매수 근거를 검증하고, 보유 이후 대응하는 각 전환점에는 서로 다른 맥락과 신뢰 장치가 필요합니다. 기존 서비스는 콘텐츠 생산에는 강했지만 발견에서 행동까지 이어지는 투자 여정을 여러 화면과 정보원으로 분리했습니다.

비밀노트는 ‘AI가 무엇을 말하는가’보다 ‘사용자가 다음 행동으로 전환하는가’를 제품의 기준으로 삼았습니다.

개선 과정 사용자 흐름 가설 → 운영 서비스 구현 → 행동 중심 개선

    외부 변수와 주가의 인과관계를 다음 의사결정의 지식으로 축적합니다.

트렌드 분석 · 신규 종목 탐색

01 / 신규 종목 탐색

트렌드 분석

신규 종목 탐색

핵심 가치거시 이슈를 개인이 탐색할 수 있는 종목 기회로 바꾸는 AI 탐색 기능입니다.
문제
이슈 콘텐츠를 읽는 것과 투자 기회를 발견하는 것은 다릅니다. 소비에서 탐색으로 이어지는 흐름이 필요했습니다.
활용 정보
경제 이벤트, 원유·환율·금리, 실시간 뉴스, 중요도, 영향 변수·업종과 대표 종목을 종합해 분석합니다.
이용 흐름
이벤트 발견 → 영향 흐름 확인 → 수혜·부담 후보 탐색 → 종목 상세 진입으로 이어집니다.
화면 구성
발표 시각, 중요도, 확인 지표와 원인·결과를 카드와 영향 흐름도로 연결해 능동적인 탐색을 유도합니다.
기대 효과
종목 발견 시간을 줄이고 익숙한 종목에 갇히는 탐색 편향을 낮춰 후보군을 구조적으로 확장합니다.

종목 상세 · AI 요약

02 / 매수 판단

종목 판단 요약

매수 판단 요약

핵심 가치7개 데이터 축을 실시간 상황과 함께 설명하는 한눈에 보는 판단 요약입니다.
문제
결론만 제시하는 AI는 신뢰하기 어렵습니다. 당일 급등, 중기 약세, 수급 매도처럼 충돌하는 신호를 같은 맥락에서 설명해야 합니다.
활용 정보
KIS 실시간 시세, 1·3개월 가격 이력, 거래대금, 외국인·기관 수급, 기업가치·실적, 뉴스 분위기를 같은 기준으로 정리합니다.
이용 흐름
후보 종목 진입 → 현재 흐름 확인 → 긍정·부정 신호 비교 → 데이터 완성도 검증으로 이어집니다.
화면 구성
핵심 신호를 색상과 언어로 구분하고 데이터 완성도를 함께 보여줘 AI 판단의 불확실성을 숨기지 않습니다.
기대 효과
핵심 파악 시간을 줄이고 단일 지표에 대한 과신을 낮춰 사용자가 판단의 맥락을 이해하도록 돕습니다.

종목 상세 · AI 전략

03 / 매수 전략

매수 타이밍 전략

매수 타이밍 전략

핵심 가치AI 분석을 관찰 가능한 행동 조건으로 바꾸고 충동적인 매수를 막습니다.
문제
분석이 다음 행동으로 이어지지 않으면 사용자는 매수 순간에 다시 감정적으로 결정합니다. 분석과 실행 사이의 간극을 줄여야 했습니다.
활용 정보
현재가·전일 종가, 변동성, 단기·중기 흐름, 거래대금 변화, 가격 분포와 분석 데이터 완성도를 사용합니다.
이용 흐름
신규 매수 보류·관찰 여부를 먼저 확인하고 관찰 가격대, 돌파 조건과 판단이 틀렸다고 볼 가격을 순서대로 검토합니다.
화면 구성
‘오늘의 판단 → 관찰 가격대 → 전환 조건 → 축소 기준’ 구조와 가격 기준선으로 긴 분석보다 실행 조건을 먼저 보여줍니다.
기대 효과
충동적인 추격 매수를 줄이고 동일한 조건에는 일관되게 대응하는 의사결정 습관을 만듭니다.

종목 상세 · 근거 검증

04 / 근거 검증

판단 근거 검증

판단 근거 검증

핵심 가치AI 판단에서 최신 데이터와 원문 출처까지 확인할 수 있는 신뢰 구조입니다.
문제
금융 규제 환경의 AI 경험은 최신성, 출처, 설명 가능성이 신뢰의 전제입니다. 모든 결론은 원천 데이터까지 확인할 수 있어야 합니다.
활용 정보
증권사 리포트 목록·상세·PDF, 목표주가, 투자의견, 발행일, DART 공시와 뉴스 원문 링크를 종목코드에 연결합니다.
이용 흐름
요약과 전략을 검토한 뒤 근거 탭에서 최신 조사 자료를 확인하고 필요한 문서는 원문까지 열어 직접 검증합니다.
화면 구성
최신 리포트 수, 목표가, 의견, 발행사, 날짜와 원문을 한 화면에서 확인하도록 정보 출처를 명확히 보여줍니다.
기대 효과
AI 판단의 설명 가능성과 신뢰를 높이고 오래되었거나 출처가 불명확한 정보가 결정에 섞이는 위험을 낮춥니다.

05 / 보유종목 대응

보유종목 대응

보유종목 대응

핵심 가치관심종목을 보유종목으로 보고 시장 상황과 대응 우선순위를 개인화합니다.
문제
모두에게 같은 시황 정보는 보유 이후의 문제를 해결하지 못합니다. 내 자산과 관련된 변화만 선별하는 개인화가 필요했습니다.
활용 정보
관심종목 구성, 실시간 시세, 외국인·기관 수급, 종목 뉴스, 주요 이벤트, 미국 지수·업종 흐름과 종목별 민감도를 사용합니다.
이용 흐름
국내 장전·장중·장마감과 미국장 상태에 따라 AI 시황 리포트를 확인하고 우선 확인할 종목부터 대응합니다.
화면 구성
시장 상태를 자동 인식하고 핵심 시황, 위험 신호, 이벤트·뉴스와 보유종목 우선순위를 개인화된 요약으로 제공합니다.
기대 효과
정보 관련성을 높여 반복 사용 가치를 만들고 중요한 변화가 생긴 종목부터 일관되게 대응하도록 돕습니다.

06 / 시장 학습

시장 영향 학습

시장 영향 학습

핵심 가치AI 답변을 시장 원리의 이해로 확장해 금융 지식을 반복해서 쌓도록 돕습니다.
문제
정답만 제공하는 AI는 의존을 만듭니다. 금리·환율·원자재 변화가 주가에 전달되는 원리를 이해해야 다음 판단이 나아집니다.
활용 정보
미국 국채금리·실질금리, 원/달러·달러지수, 장단기 금리차, 변동성 지수, 원유·구리, 나스닥과 위험자산 지표를 사용합니다.
이용 흐름
다섯 외부 요인의 방향과 비중을 보고 원인과 결과, 영향 업종, 대표 종목과 공식 지표를 차례로 확인합니다.
화면 구성
요인 지도, ‘외부 변수 → 국내증시 → 영향 종목’ 흐름과 학습 내용을 결합하고 데이터 연결 상태도 투명하게 표시합니다.
기대 효과
콘텐츠 소비를 금융 지식의 축적으로 연결해 장기적인 신뢰와 자기주도적 판단 역량을 만듭니다.
