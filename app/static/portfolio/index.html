THE PROBLEM

정보는 많지만,
지금 해야 할 행동은 보이지 않았다.

개인 투자자는 시세 앱, 뉴스, 공시, 증권사 PDF와 미국 시장을 따로 확인합니다. 정보는 계속 늘어나지만 내 종목에 어떤 영향을 주는지, 지금 무엇을 모니터링해야 하는지는 스스로 다시 해석해야 합니다.

비밀노트는 이 해석 비용을 줄이는 데서 출발했습니다.

관심종목 · AI 시황 리포트

01 / CONTEXT

AI 시황 리포트

시장의 사건을 내 종목의 모니터링 우선순위로 바꿉니다.

“미국장만 포커싱하지 말고 주요 이벤트, 뉴스 등 시황을 종합해서 내 종목 모니터링 리포트로 보여줘.” 실제 사용자 요청 어투
서비스 기능
거시 이벤트, 뉴스 분위기, 미국 섹터 흐름과 국내 수급을 결합해 관심종목별 영향 신호와 확인 우선순위를 생성합니다.
유저 저니 목적
장 시작 전후에 여러 앱을 이동하지 않고, 오늘 내 종목에 중요한 변수부터 빠르게 확인합니다.
기대 효과
시황 탐색 시간을 줄이고, 시장 이슈와 무관한 종목까지 불필요하게 반응하는 행동을 줄입니다.
  • 이벤트·뉴스 의미 분류
  • 종목 영향도 매칭
  • 모니터링 우선순위 생성

종목 상세 · AI 요약

02 / SYNTHESIS

AI 종목 판단

서로 다른 지표를 하나의 일관된 판단 언어로 번역합니다.

“실제 주가는 엄청 상승하고 있는데 상승 가능성은 낮게 보여. 현재 흐름과 판단의 괴리가 없도록 봐줘.” 실제 사용자 요청 어투
서비스 기능
실시간 시세, 단기·중기 모멘텀, 수급, 밸류에이션, 뉴스 감성을 함께 계산하고 긍정·부정 신호를 색과 문장으로 요약합니다.
유저 저니 목적
주가 상승 하나만 보고 추격하지 않고, 상승을 지지하거나 경고하는 신호를 함께 비교합니다.
기대 효과
단일 지표의 과신을 줄이고, 판단의 데이터 완성도와 현재 시장 국면을 한눈에 파악합니다.
  • 7개 데이터 축 정규화
  • 장중 맥락 반영
  • 신호 충돌 설명

종목 상세 · AI 전략

03 / ACTION

행동 전략

분석 결과를 사용자가 바로 이해할 수 있는 대응 순서로 바꿉니다.

“정보가 너무 많고 어쩌라는 건지 모르겠어. 지금 해야 할 행동이 먼저 보였으면 좋겠어.” 실제 사용자 요청 어투
서비스 기능
현재가와 변동성 데이터를 기반으로 관찰 가격대, 전환 조건, 비중 축소 기준과 이익 관리 구간을 단계별로 제시합니다.
유저 저니 목적
긴 분석을 모두 읽기 전에 오늘의 대응과 핵심 가격을 먼저 확인하고, 필요할 때만 상세 근거를 펼칩니다.
기대 효과
충동적인 추격 매수와 기준 없는 손절을 줄이고, 사전에 정한 조건에 따라 일관되게 대응합니다.
  • 행동 우선 정보 구조
  • 동적 가격 시나리오
  • 근거 점진 공개

종목 상세 · 근거 검증

04 / EVIDENCE

근거 검증

AI의 결론 뒤에 있는 데이터와 원문을 사용자가 직접 확인합니다.

“증권사 리포트가 발행되면 자동으로 쌓이고, 우리가 만든 웹과 모바일에서 바로 보고 싶어.” 실제 사용자 요청 어투
서비스 기능
증권사 리포트를 주기적으로 수집하고 종목코드에 연결해, 목표가·투자의견·발행일·원문을 판단 근거 탭에서 제공합니다.
유저 저니 목적
AI 요약을 그대로 믿는 대신, 최신 리서치와 뉴스 원문을 열어 결론의 최신성과 출처를 검증합니다.
기대 효과
AI 분석의 설명 가능성과 신뢰를 높이고, 과거 자료나 출처 없는 결론이 투자 판단에 섞이는 것을 줄입니다.
  • 5분 증분 수집
  • 일일 과거자료 보강
  • 종목·리포트 자동 연결
