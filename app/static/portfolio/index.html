문제 발견

정보는 많지만,
판단은 어려웠습니다.

투자자는 뉴스, 시세, 공시와 증권사 리포트를 여러 곳에서 따로 확인해야 합니다. 정보가 많아도 어떤 종목을 먼저 보고, 지금 사도 되는지, 보유 종목에 어떻게 대응해야 하는지는 직접 판단해야 했습니다.

비밀노트는 흩어진 정보를 한곳에 모아, 지금 무엇을 먼저 확인하고 어떻게 대응할지 쉽게 보여줍니다.

사용 흐름

매일 확인하는
네 단계

  1. 01 · 대응오늘 볼 종목을 확인합니다

    보유 종목 중 먼저 살펴볼 종목과 그 이유를 알려줍니다.

  2. 02 · 찾기새 종목을 찾습니다

    시장 이슈와 관련된 업종과 종목을 함께 보여줍니다.

  3. 03 · 확인종목 상태를 확인합니다

    주가, 수급, 실적, 뉴스와 리포트를 한눈에 비교합니다.

  4. 04 · 이해시장 흐름을 이해합니다

    금리, 환율과 원자재가 주가에 미치는 이유를 설명합니다.

★ 01 / 대응 정보

보유 종목 대응하기

오늘 먼저 확인할 종목과 이유 안내

★
매일 들어와야 하는 이유

시세·수급·공시·뉴스가 계속 바뀌기 때문에 오늘 먼저 확인할 종목과 대응 순서도 매일 달라집니다.

한눈에 보기현재 시장 상황이 내 종목에 미치는 영향과 오늘 먼저 볼 종목을 알려줍니다.
해결한 문제
전체 시장 뉴스가 내 보유 종목에 좋은지 나쁜지 직접 연결해서 판단하기 어려웠습니다.
사용한 정보
관심종목, 실시간 주가, 외국인·기관 매매, 종목 뉴스, 주요 일정과 미국 시장 흐름을 사용합니다.
사용 순서
오늘의 시장 변화 확인 → 내 종목에 미치는 영향 확인 → 먼저 볼 종목 확인 순서로 봅니다.
보여주는 방법
핵심 시장 소식, 좋은 신호와 나쁜 신호, 확인이 필요한 종목 순위를 짧게 보여줍니다.
기대 효과
모든 종목을 확인하지 않아도 변화가 큰 종목부터 빠르게 대응하고, 매일의 투자 점검을 놓치지 않을 수 있습니다.

트렌드 분석 · 신규 종목 탐색

02 / 종목 찾기

새 종목 찾기

시장 이슈와 관련 종목 확인

한눈에 보기중요한 시장 소식과 관련 종목을 함께 보여줍니다.
해결한 문제
뉴스를 읽어도 어떤 업종과 종목이 영향을 받는지 바로 알기 어려웠습니다.
사용한 정보
경제 일정, 원유, 환율, 금리, 실시간 뉴스, 관련 업종과 대표 종목을 사용합니다.
사용 순서
중요 이벤트 확인 → 관련 업종 확인 → 오를 수 있는 종목과 부담을 받을 종목 확인 순서로 봅니다.
보여주는 방법
발표 시간, 중요도, 관련 업종과 종목을 한 화면에서 연결해 보여줍니다.
기대 효과
시장 이슈와 연결된 새 종목을 더 빠르게 찾을 수 있습니다.

종목 상세 · AI 요약

03 / 상태 확인

종목 상태 한눈에 보기

주요 정보를 한 화면에서 비교

한눈에 보기현재 주가와 지난 흐름, 수급, 실적, 뉴스를 한곳에서 보여줍니다.
해결한 문제
오늘 주가가 올라도 지난 흐름이나 수급이 나쁠 수 있어 한 가지 숫자만으로 판단하기 어렵습니다.
사용한 정보
실시간 주가, 1·3개월 등락률, 거래대금, 외국인·기관 매매, 실적과 뉴스 분위기를 사용합니다.
사용 순서
현재 주가 확인 → 좋은 신호와 나쁜 신호 비교 → 빠진 정보가 없는지 확인 순서로 봅니다.
보여주는 방법
좋은 신호와 나쁜 신호를 색으로 구분하고, 분석에 사용한 정보의 개수도 함께 보여줍니다.
기대 효과
종목의 현재 상태와 주의할 점을 짧은 시간에 파악할 수 있습니다.

종목 상세 · AI 전략

04 / 매수 판단

매수 시점 판단하기

지금 살지 기다릴지 확인

한눈에 보기지금 매수할지, 기다릴지, 어느 가격을 확인할지 알려줍니다.
해결한 문제
분석을 읽고도 실제 매수 순간에는 감정에 따라 결정하기 쉽습니다.
사용한 정보
현재가, 전일 종가, 가격 변동폭, 1·3개월 흐름과 거래대금 변화를 사용합니다.
사용 순서
오늘의 판단 확인 → 기다릴 가격 확인 → 매수 검토 조건 → 손실을 줄일 기준 순서로 봅니다.
보여주는 방법
긴 설명보다 기다릴 가격, 매수 검토 가격과 주의 가격을 먼저 보여줍니다.
기대 효과
급하게 따라 사는 행동을 줄이고 미리 정한 기준으로 판단할 수 있습니다.

종목 상세 · 근거 검증

05 / 자료 확인

판단 근거 확인하기

리포트와 원문 직접 확인

한눈에 보기AI가 참고한 증권사 리포트, 공시와 뉴스 원문을 직접 확인할 수 있습니다.
해결한 문제
AI의 결론만 보면 어떤 자료를 참고했는지, 자료가 최신인지 알기 어렵습니다.
사용한 정보
증권사 리포트, 목표주가, 투자의견, 발행일, 공시와 뉴스 원문을 사용합니다.
사용 순서
AI 요약 확인 → 최신 리포트 확인 → 필요한 자료의 원문 열기 순서로 봅니다.
보여주는 방법
리포트 수, 목표주가, 의견, 증권사, 날짜와 원문 링크를 한 화면에 보여줍니다.
기대 효과
정보의 출처와 날짜를 직접 확인해 더 안심하고 판단할 수 있습니다.

시장 영향도 분석 · 금융 지식 학습

06 / 시장 이해

시장 흐름 이해하기

주가가 움직이는 이유 설명

한눈에 보기금리, 환율과 원자재 변화가 주식시장에 미치는 이유를 쉽게 설명합니다.
해결한 문제
금리나 환율이 올랐다는 뉴스는 많지만 내 종목에 왜 영향을 주는지 이해하기 어려웠습니다.
사용한 정보
미국 금리, 원/달러 환율, 달러 가치, 원유, 구리, 나스닥과 시장 불안 지표를 사용합니다.
사용 순서
변화가 큰 요인 확인 → 국내시장 영향 확인 → 영향을 받는 업종과 종목 확인 순서로 봅니다.
보여주는 방법
‘외부 변화 → 국내시장 → 관련 종목’ 순서로 원인과 결과를 연결해 보여줍니다.
기대 효과
시장 뉴스를 이해하고 다음 투자 판단에 활용할 수 있는 기초 지식을 쌓을 수 있습니다.
